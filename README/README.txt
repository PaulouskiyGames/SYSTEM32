Суть игры просто передвигаться по уровню и отбиватьсыя от волн противников при помощи меча. В игре реализованна механика передвижения с разной скоростью, атака, смерть персонажа, его анимации. Враги спавняться рандомно в заранее созданный точках спавна и сразу бегут за игроком, чтоб атаковать. При смерти игрок физически падает при помощи Ragdoll анимации и отправляется в пул объектов ждать своей очереди на респавн. Игра длиться пока игрок не умрет. В игре реализованна пауза. Также добавленны минимально звуки.
